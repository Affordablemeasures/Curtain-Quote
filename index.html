📋 Quoting Tool — Help & Instructions

Affordable Curtains & Blinds · Internal use only

Welcome to the Quoting Tool

This tool generates curtain quotes for Affordable Curtains & Blinds. It handles two modes — Detailed Quote for full pricing, and Quick Quote for fast ballpark estimates. Both modes use the same Norfolk price grid for make costs.

💡 All prices are retail inc. GST unless the breakdown shows "(ex-GST)" or "(wholesale)".

First Steps

1 — Select your name

On launch the tool asks who is quoting. Select your name to set your initials — these are included in the quote number (e.g. CU-ST-13-04-2026-01). You can change who is quoting at any time by clicking the initials badge in the top-left.

2 — Fill in customer details

Enter the customer's name, address, phone, and email. The Date of Measure and Quote Date are separate — fill in both. Click Generate Quote # to create a new quote number.

3 — Choose your mode

Use the toggle in the header to switch between 📋 Detailed Quote and ⚡ Quick Quote. Detailed is for full quoted jobs. Quick Quote is for initial estimates when you don't yet have all the fabric details.

💡 The Fuel / Global Surcharge field applies a percentage on top of the entire quote total. Leave at 0% if not applicable.

Quote Numbering

Quote numbers are formatted as CU-[INITIALS]-[DD-MM-YYYY]-[NN]. The counter resets daily per staff member, so each person can have their own sequence each day. Numbers are stored in the browser and persist across sessions.

Adding Windows

Click + Add Window to add a new window card. Each window is independent. You can have as many windows as needed — they all roll up into the grand total.

Curtain Type

Select the curtain type first — Pencil Pleat, Wave Fold, or Pinch/SPP Pleat. The form sections that appear depend on the type selected. Wave Fold uses a separate set of fields and fabric measurements.

Track & Bracket

Select the track type and bracket from the dropdowns. The tool looks up the retail price from its built-in price lists. If a price list hasn't been loaded for a track type yet, it will say "No price list loaded — supply manually".

💡 When you select Wave Fold (Series 51), the tool automatically applies wave fold defaults and adds Flick Sticks to the pricing. Adjust quantity and size as needed.

Track Width

You can either type the track width directly, or use the Opening + Past Left + Past Right fields to calculate it. Both methods update the same track width value.

Measurements

  • Track / Rod Width — the finished width of the track in mm.
  • Required Drop — the measured drop in mm from the top of the track to the floor (or desired hem position).
  • Drop Allowance — calculated automatically based on curtain type, heading tape size, and weighted base selection. Do not enter manually.
  • Cut Drop — automatically calculated. This is what the workroom cuts to.
⚠ For continuous fabric, you must confirm the roll width before the calculation runs. Tick the confirmation checkbox once you've verified the width with the supplier.

Fabric Details

Enter the Supplier, Range, Colour, and Fabric Cost (ex-GST per metre). The retail price is automatically calculated at 100% markup. Fabric category (Blockout Coated, Sheer, Uncoated, Triple Weave) affects the default lining behaviour.

Fullness

Select the fullness multiplier. Default is 2.25× for most curtains. Sheers typically use 2.5×. You can set a custom multiplier if needed.

Lining

Tick Include loose lining to add a separate lining make cost and fabric cost. Enter the lining fabric cost per metre. The lining uses the same meterage as the face fabric.

Make Cost

Make cost is looked up automatically from the Norfolk price grid based on the track width and drop. The grid uses the wholesale price, which is then marked up at 60% to get the retail make cost. No discount is applied.

Quote A / Quote B

The tool supports two quote variants (A and B) for the same customer — useful when offering a fabric choice or a premium vs. budget option. Use the tabs above the window list to switch between quotes. Quote B is a copy of Quote A that you can then modify independently.

Promotions

The Promotions dropdown in Global Pricing Controls lets you apply a discount to the entire quote:

  • 10% off total — applies to everything including install.
  • 10% off uninstalled — applies to fabric, make, and track only.
  • Free install — removes all install charges from the total.

Quick Quote Mode

Quick Quote is designed for fast ballpark estimates — when you're on the phone or at a measure and need a rough number quickly. It uses simplified assumptions and does not require fabric cost details.

Global Settings

  • Default Curtain Type — sets the type used for all windows unless overridden per window.
  • Fabric Cost — a single assumed cost per metre used across all windows.
  • Fullness — applies to all windows unless overridden.
  • Include Install — toggle whether install is included in the estimate. Default is ON (timber fix, standard height).
  • Install Type — sets the default install type for the estimate.
💡 The Assumptions banner at the top of Quick Quote mode shows exactly what is being assumed for this estimate.

Per-Window Overrides

Each window card has an ⚙ Override defaults section where you can change the track type, bracket, fullness, bend type, and install per window.

Accuracy

Quick Quote uses continuous 3m fabric for all calculations, assumes centre-open hang, 100mm returns, and 200mm drop allowance. It is an estimate only — always confirm with a Detailed Quote before ordering.

Install Rates

Install is calculated per window based on the number of hanging drops, the drop height tier, the track fix, and the install type.

Drop Height Tiers

  • ≤2700mm — standard rate, $25.00 per drop
  • 2701mm – 4000mm — elevated rate, $50.00 per drop
  • >4000mm — elevated rate ($50/drop) plus a scaffold surcharge field appears. Enter the agreed POA amount. Note: may require scaffold.

Install Types

  • Timber fix, standard height — $40 track fix + hang rate per drop
  • Plaster / brick / steel, standard height — $80 track fix + hang rate per drop
  • High fit / 2-man — $80 track fix + $50 per drop (regardless of measured drop)
  • Clear window — leave on site — $10 flat per window
  • Clear window — dispose — $20 flat per window

Hanging Drops Calculation

The number of hanging drops is calculated from the required fabric width (track width × fullness) ÷ 1500mm, then rounded up. For centre-open curtains the result is rounded to the next even number. For one-way draw it is rounded to the next whole number.

💡 You can override the install cost manually on any window. Type the override amount and it replaces the calculated value. Click ✕ Clear to revert to calculated.

Markup Summary

  • Fabric — 100% markup on cost price (cost × 2 = retail)
  • Make cost — 60% markup on Norfolk wholesale (wholesale × 1.6 = retail)
  • Track — 100% markup on wholesale price (wholesale × 2 = retail)
  • Bends — 100% markup on $27.50 wholesale per bend
  • All prices rounded up to the nearest $5.00

Fabric Library

The Fabric Library builds up automatically as you quote. Every time you enter a fabric with a supplier, range, and cost, it is saved to the library for future use.

Auto-complete

As you type in the Supplier, Range, or Colour fields, matching fabrics from the library appear in a dropdown. Click a result to instantly fill in all fabric fields — supplier, range, colour, cost, category, and width type.

Auto-save

Fabrics are saved automatically when a cost is entered and the supplier + range fields are filled. If you quote the same fabric at a different price, the stored price updates automatically.

Managing the library

Open the 📚 Fabric Library panel at the bottom of the page to browse, search, or delete saved fabrics. Each entry shows how many times it has been used. Click ✕ on any row to remove it from the library.

💡 The library is stored in your browser's local storage and persists across sessions. It is specific to the computer and browser you are using — it does not sync between machines.
⚠ Clearing your browser's site data or using a different browser will clear the fabric library. Export/backup coming in a future version.

Corner & Bay Window Bends

Select the window shape from the Window Shape dropdown in the Bends section. A diagram modal opens automatically for bay, corner, and box bay windows.

Corner Window — 1 Bend

Choose Diagram A or Diagram B to set the corner orientation:

  • Diagram A — left section runs down the wall, right section runs along the top
  • Diagram B — right section runs down the wall, left section runs along the top

Enter the width of each section. The diagram shows Left section and Right section labels.

Bay Window — 2 Bends (Angled)

Enter the width of the left wing, centre, and right wing sections separately. The total is used as the track width for pricing.

Box Bay — 2 Bends (Square/90°)

Same as bay window but the wings are at 90° to the centre. Enter each section width.

Custom Bends

For unusual window shapes, select Custom. Enter the number of bends and the total combined track width. Attach a diagram sketch to the job file — the factory works from the diagram for unusual shapes. The quote is priced on the bends and total width entered.

Bend Size Type

In the diagram modal, select how your measurements are supplied to the factory:

  • Actual bend sizes — the sizes you enter are the exact finished dimensions the factory bends to.
  • Allow for bracket projection — your sizes are the measured dimensions; the factory adjusts for bracket projection. Choose Forward (factory adds projection forward of your sizes) or Backward (factory deducts from your sizes). The quote price is always based on your entered sizes — this is a note to the factory only.
💡 Bend pricing uses the Series 51 SAN price list at $27.50 wholesale per bend. Cord-drawn Wave Fold tracks cannot be bent.

Saving a Quote

Click 💾 Save in the header to download the quote as a .json file. The filename includes the quote number and customer name (e.g. CU-ST-13-04-2026-01_Smith_Jane.json).

To save to Google Drive: after downloading, open Google Drive in your browser and drag the file into your Curtain Quotes folder.

Loading a Quote

Click 📂 Load to reload a previously saved .json file. This restores customer details and settings. Window data from saved quotes is planned for a future version.

New Quote

Click 💾 New Quote to save the current quote and start a fresh blank quote. The tool will ask you to confirm before clearing.

Printing & PDF

Click 🖨 Print/PDF to open the browser print dialog. The tool includes a print-optimised stylesheet — headers, totals bars, and dark backgrounds are automatically adjusted for paper.

To save as a PDF: in the print dialog, change the destination to Save as PDF. Set paper size to A4 and orientation to Landscape for the job sheet summary.

💡 The Job Sheet at the bottom of the Detailed Quote is designed for A4 landscape — it includes all window details in a compact table suitable for workroom handover.

Staff List

The staff list is managed via the ⚙ Edit staff list button in the Who is quoting? modal. Format each entry on its own line as:

Full Name, XX — where XX is 2–4 letters for initials.

Example: Jane Smith, JS

Changes save immediately and take effect on the next quote number generation.
